Mosaic Plot

Great At:

  • Designed to create visualizations of discrete data.
  • Works well for combinations that are between 4 and 24.
  • Helpful for displaying raw data and they can also be used to support modelling
  • Variables divided up into rectangles proportional in size to the counts of the combinations they represent
  • For a limited number of variables with only a few categories, mosaicplots can work well, although they need careful design.

    • Tips:

  • Labeling
      • Difficult at times.
  • Variables
      • The first variable following the tilde is dependent, all the other variables are independent.
  • Looking for features:
      • Probability

    Code:

      library(ggplot2)
      library(GGally)
      mosaicplot(~ gear + carb, data = mtcars, color = 4:5)
