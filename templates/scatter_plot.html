Scatter Plot

Great At:

  • Discovering associations between variables
  • Finding causal Relationships/Correlations
  • Identifing outliers
  • Seeing gaps in data
  • Seeing clusters of groups
  • Spotting distribution features

    • Tips:

  • Point Size
      • Small points are hardly seen and big points cover other points making it hard to read the chart.
      Control the size of the points and make sure it's not too large or too small.
  • Symbols
      • Use different shapes for different groups when creating a scatter plot.
      Use your judgement in how many symbols make sense before the plot takes a dive toward looking like a mess.
  • Alpha-blending
      • The points are made slightly opaque on a scale of 0 to 1.
      This allows to see overplotting issues in graphs.
  • Smoothing
      • Smoothes the scatter plot by fitting a line to the data points
  • Coloring points
      • Like "Symbols", "Coloring" also is used to distinguish different categories

    Code:

    library(ggplot2)
    df <- mtcars
    ggplot(df, aes(x=mpg, y=wt)) +
      geom_point(alpha=.9, aes(size = 3, colour = factor(am), shape = factor(cyl))) +
      geom_smooth(method=lm , color="red")
