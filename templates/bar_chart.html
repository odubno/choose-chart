Bar Chart

Great At:

  • Great a displaying categorial/discrete variables
  • Has gaps between bars, because the data is not continuous. This being one of the differences between histograms and bar charts.
  • Area of bar is represented by it's group size.
  • Give a good relative size of your categories.
  • Great for opinion polls, questionnaires and surveys.
  • Barcharts can be used for nominal, ordinal, or discrete variables.
  • The order of categories affects the way a barchart of a nominal variable looks, and different orderings can emphasise different features

    • Tips:

  • Labeling
      • Difficult at times.
  • Bins
      • Does not use it. Does not need it.
  • Looking for features:
      • Relationships
      Patterns
  • Top Coding
      • At times, top coded data is in place to keep the anonymity of data.
      The last value has a “+” category to show that values still continue going up.
      There’s a large bin at the end to show that there’s a possibility of a lot more data.
      Top coding is a problem if there's a lot of data being hidden.

    Code:

    library(ggplot2)
    ggplot(mpg, aes(class)) +
      geom_bar(aes(fill = drv), position = position_stack(reverse = TRUE)) +
      coord_flip() +
      theme(legend.position = "top")
