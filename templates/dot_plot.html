Dot Plot

Great At:

  • A dot over each data value.
  • Dots are stacked, with each dot representing one observation.
  • Include ALL values from the data set, with one dot for each occurrence of an observed value.
  • Good for gaps, although not for heaping.
  • Poor for large datasets.

    • Tips:

  • Bins
      • The width of a dot corresponds to the bin width.
      Default is 30. Use binwidth to override.
      Always explore other bin widths and find the best one to showcase your data.
      Good general rule is to use the square root of the number of observations (round up the result).
  • Dotsize
      • The diameter of the dots relative to binwidth, default 1.
  • Looking for features:
      • Asymmetry
      Outliers
      Multimodality
      Gaps
      Impossibilities / Errors

    Code:

    library("ggplot2") df <- mtcars ggplot(df, aes(x = mpg)) +
      geom_dotplot(binwidth = 1.5, color='white', fill='red')
