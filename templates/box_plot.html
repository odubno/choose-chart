Box Plot

Great At:

  • Great summary of the variable.
  • The line that divides the box into 2 parts represents the median.
  • Display statistical information such as.
      • minimum, lower-hinge (lower quartile)
      median
      upper-hinge (upper quartile)
      maximum
      individual outliers for the data
  • Extremely useful for identifying outliers, comparing quartiles and for comparisons of distributions across subgroups.
  • Obscure multimodality. Makes it difficult to identify modes.
  • Boundaries could be right close or right open.
      • left-open, right-open: (4, 8) 4 < x < 8
      left-closed, right-open: [4, 8) 4 ≤ x < 8
      left-open, right-closed: (4, 8] 4 < x ≤ 8
      left-closed, right-closed: (4, 8] 4 ≤ x ≤ 8
  • Unlike a barchart, histograms have no gaps between intervals.
  • Spotting distribution features

    • Tips:

  • Quartiles
      • Reorder the subgroups by median, maximum value or standard deviation rather than initially alphabetical ordering could make boxplots looks nicer.
  • Outliers
      • Susptected outliers and outliers are above 3rd quartile and below 1st quartile.
      Outlier are not necessarily bad.
      Eg. if the shape of the histogram stays the same as bin width is varied, there's rounding errors.
  • Variables
      • Use variable width boxplot to show the amount of data in each subplot, while wider box width represents more data.
  • Multiple Box Plots
      • The boxplots are not to be compared with one another, drawing them all together in one plot is primarily a time and space saving exercise.
  • Looking for features:
      • Outliers
      Gaps
      Impossibilities / Errors
